# Skills Overview

This folder contains small, practical skill prompts for Google AI Edge Gallery + Gemma local usage.

## Principles

Every skill here should be:

- single-purpose
- easy to copy
- structured in output
- useful on mobile
- testable in under 10 minutes

## Start here

- `story-hook-generator.md` → for creators testing short-form hooks
- `storyboard-6shots.md` → for visual planning
- `readme-rebuilder.md` → for GitHub cleanup
- `local-workflow-planner.md` → for execution clarity

## How to improve a skill

If a skill feels weak, do not make it longer first.
Fix it in this order:

1. tighten the input fields
2. tighten the output format
3. remove vague wording
4. add a better success check
