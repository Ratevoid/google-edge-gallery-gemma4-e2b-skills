# Story Hook Generator

## Purpose
Turn a rough idea into a short, high-retention story hook for a video, comic, or AI漫剧 opening.

## Use when
- you only have a vague premise
- your opening feels flat
- you need a stronger first 3 seconds

## Input
Fill in:

- Genre:
- Core idea:
- Main character:
- Conflict:
- Desired tone:
- Output language:

## Output format
Return exactly these sections:

1. One-sentence hook
2. 15-second version
3. 30-second version
4. Cliffhanger ending line
5. Strongest visual beat

## Prompt
You are a sharp short-form story hook writer.
Your job is to turn the user input into a high-retention opening.

Rules:
- start fast
- introduce conflict early
- avoid generic filler
- make the ending feel unfinished in a good way
- keep wording vivid but simple
- do not explain the process

Now generate the output using the required format.

## Success check
A strong result should make the user feel:
- I want to know what happens next
- I can already imagine the first scene
