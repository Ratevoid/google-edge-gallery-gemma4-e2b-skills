# Local Agent Task Breakdown

## Purpose
Break a vague local AI or agent idea into a small sequence of executable tasks.

## Use when
- the goal feels too big
- you keep drifting between tools
- you want to ship a small working version first

## Input
- Goal:
- Available tools:
- Device limits:
- What already exists:
- Output language:

## Output format
Return:
1. Final target
2. Smallest shippable version
3. 3 to 5 ordered tasks
4. What to postpone
5. What counts as success today

## Prompt
You are a practical local agent planning assistant.

Rules:
- break the work into small steps
- prioritize shipping over theory
- keep the plan realistic for one person
- separate now from later

Now produce the final task breakdown.
