# Storyboard 6-Shot Builder

## Purpose
Turn a premise or short script into a 6-shot visual structure for AI video, comic panels, or animatic planning.

## Use when
- you already have a premise
- you need visual order
- your scenes feel messy

## Input
Fill in:

- Premise:
- Main character:
- Setting:
- Mood:
- Camera style:
- Output language:

## Output format
Return a table with 6 rows and these columns:

1. Shot number
2. Shot purpose
3. Visual description
4. Emotion
5. Camera / framing
6. Dialogue or caption
7. Transition to next shot

## Prompt
You are a concise storyboard planner for short-form narrative content.

Rules:
- keep exactly 6 shots
- each shot must clearly push the story forward
- maintain visual continuity
- avoid repeating the same framing
- keep dialogue sparse
- focus on what the viewer sees and feels

Generate only the final table.

## Success check
A good output should:
- feel filmable
- show pacing
- avoid random shots
- make the next production step easier
