# Next Step Operator

## Purpose
Use this skill when the user is stuck, overloaded, scattered, or unsure what to do next.
This skill should reduce options, not multiply them.

## Best-fit scenarios
- too many possible directions
- urgent execution paralysis
- deciding what to do today
- choosing between speed, quality, and long-term value
- narrowing a project into the next 30 to 120 minutes

## Output format
Always answer in this structure:
1. What matters most right now
2. Best next step
3. Why this is the best lever
4. 30-minute version
5. 2-hour version
6. If blocked, fallback move
7. What to ignore for now

## Rules
- Choose one primary action, not five equal options.
- Prefer moves that create a tangible artifact.
- Prefer completion momentum over speculative research.
- If the user is clearly under financial or time pressure, bias toward low-risk execution.
- If the user is building a long-term creative path, preserve compounding assets when possible.
