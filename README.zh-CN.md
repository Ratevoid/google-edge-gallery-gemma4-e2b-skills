# Google Edge Gallery Gemma 4 E2B 技能包

这是一个面向 **Google AI Edge Gallery** 与 **Gemma 本地使用场景** 的实用技能仓库。

它不是模型跑分仓库，也不是一堆空泛提示词合集。
它更像一个可以直接复制、测试、改造的 **本地 AI 技能包**。

## 这个仓库适合谁

- 想在本地设备上把 Gemma 用起来的人
- 想做 AI 漫剧、短内容、分镜规划的创作者
- 想把空仓库整理成更像项目的独立开发者
- 想把 prompt 从“能用”推进到“稳定可复用”的本地 AI 玩家

## 目前包含什么

### 内容创作类
- 故事钩子生成
- 6 镜头分镜规划

### 仓库与开发类
- README 重写
- 本地工作流规划
- Prompt 调试
- 输出格式约束

## 推荐阅读顺序

1. `skills/README.md`
2. `docs/quick-start.md`
3. `examples/README.md`
4. `docs/use-cases.md`

## 贡献入口

如果你想补一个新 skill，先看：

- `CONTRIBUTING.md`
- `docs/skill-authoring-guide.md`
- `templates/skill-template.md`

## 这个仓库的方向

目标不是做大而全，而是做出一组：

- 小而实用
- 可快速测试
- 结构稳定
- 适合本地设备
- 适合被别人继续贡献的技能

## 当前状态

仓库正在从“空标题仓库”升级成一个真正可用、可扩展、可贡献的本地 AI 技能项目。
