# Roadmap

## Current focus
- build a small but high-signal skill set
- make the repo easy to understand in under 3 minutes
- add practical examples instead of theory-heavy notes

## Near term
- replace the placeholder root README with a full landing page
- add real screenshots from Google AI Edge Gallery usage
- add more examples for developer-facing skills
- tighten skill naming and folder structure

## Skill areas to expand
- repo cleanup and developer workflow
- prompt debugging and output control
- story planning and creator workflows
- local agent task breakdown
- structured local note or summary workflows

## Contribution goals
- make it easy for contributors to add one skill at a time
- keep every skill small, testable, and practical
- prefer repeatable structure over giant prompt dumps

## Long term direction
Turn this repo into a practical local-first skill pack for Gemma users, with:
- reusable skills
- before/after examples
- mobile-friendly testing
- clear contribution standards
