# Screenshots

This folder is for real screenshots from Google AI Edge Gallery or related local Gemma workflows.

## Recommended screenshots

- one screenshot of a skill being tested in Prompt Lab or chat
- one screenshot of a structured output that looks useful
- one screenshot showing before/after improvement if possible

## Why screenshots matter

Screenshots make this repo easier to trust.
They help visitors understand that these skills are meant to be used, not just collected.
