# Examples

This folder shows small example inputs and outputs for the skills in this repo.

## Included examples

- `story-hook-example.md` → how a vague premise becomes a stronger opening hook
- `storyboard-example.md` → how a premise becomes a 6-shot visual sequence
- `readme-example.md` → how an empty project becomes easier to understand
- `local-workflow-example.md` → how a messy local-AI goal becomes a small execution plan
- `prompt-debugger-example.md` → how a weak prompt is tightened
- `output-format-example.md` → how drifting output is pushed into a stable structure

## How to use these examples

1. Read the example input
2. Compare it with the example output
3. Test the same skill with your own input
4. Tighten the skill if the output drifts too much

## Why examples matter here

This repo is not meant to be a theory collection.
Examples make the skills easier to trust, compare, and improve.
