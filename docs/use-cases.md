# Use Cases

This repo is most useful for people who want practical local-AI output, fast.

## 1. Short-form creators
Use `story-hook-generator.md` to turn a rough premise into a stronger opening.

## 2. AI comic or video planners
Use `storyboard-6shots.md` to move from idea to visual order quickly.

## 3. Solo builders with weak repos
Use `readme-rebuilder.md` when a project has code but no readable entry point.

## 4. Local AI experimenters
Use `local-workflow-planner.md` to stop drifting between tools and choose one next action.

## 5. Prompt engineers and local model tinkerers
Use `prompt-debugger.md` and `output-format-enforcer.md` to make outputs more stable and reusable.

## What this repo is not

It is not trying to be:
- a benchmark collection
- a giant prompt dump
- a model leaderboard
- a theory-heavy prompt engineering course

It is a practical skill pack.
