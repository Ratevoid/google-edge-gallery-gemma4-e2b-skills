# Quick Start

This repo is designed for fast testing, not setup marathons.

## 1. Pick one skill
Start with one of these:

- `skills/story-hook-generator.md`
- `skills/storyboard-6shots.md`
- `skills/readme-rebuilder.md`
- `skills/local-workflow-planner.md`

## 2. Copy the prompt
Open the skill file and copy the prompt plus its input fields.

## 3. Paste into your local AI tool
Use it in Google AI Edge Gallery, Prompt Lab, or a similar local Gemma workflow.

## 4. Run the same skill 3 times
Use 3 different inputs.

A good skill should:
- stay structured
- stay useful
- not require rewriting the whole prompt each time

## 5. Save one strong result
If one output is clearly useful, save it into `examples/`.
