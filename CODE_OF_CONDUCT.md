# Code of Conduct

## Our goal

This repo is meant to be a practical, welcoming place for people building useful local-AI skills.

## Expected behavior

Please:
- be respectful
- be clear
- focus on improving the work
- give concrete feedback
- assume good intent when possible

## Unacceptable behavior

Please avoid:
- personal attacks
- harassment
- hostile or mocking feedback
- dismissive gatekeeping
- spam or low-effort disruption

## Contribution style

This project values:
- usefulness over ego
- clarity over jargon
- repeatability over hype

## Enforcement

Project maintainers may remove comments, issues, or contributions that clearly violate these guidelines.
