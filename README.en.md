# Google Edge Gallery Gemma 4 E2B Skills

A practical skill pack for using Gemma locally in Google AI Edge Gallery.

This repo is not meant to be a benchmark dump or a random prompt pile.
It is meant to be a reusable, local-first collection of small skills that actually help people get work done.

## Who this repo is for

- creators testing story hooks and storyboards
- solo builders improving weak public repos
- local AI users who want structure instead of prompt chaos
- developers who want reusable Gemma-style task prompts

## What is inside

### Creator-facing skills
- story hook generation
- 6-shot storyboard planning

### Developer-facing skills
- README rebuilding
- local workflow planning
- prompt debugging
- output format enforcement
